# Copilot-Specific Instructions

## Focus Areas

GitHub Copilot CLI is optimized for GitHub-integrated workflows: pull requests, issues, code review, and repository operations.

## Custom Instructions

This file (`~/.copilot/copilot-instructions.md`) provides persistent custom instructions loaded for every Copilot CLI session.

## Skills

Agent skills live in `~/.copilot/skills/`. Each skill directory contains a `SKILL.md` file.

## Custom Agents

Custom agent profiles live in `~/.copilot/agents/` as `.agent.md` files with YAML frontmatter (name, description, prompt fields).
